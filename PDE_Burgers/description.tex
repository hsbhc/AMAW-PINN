$$Burgers' equation$$
$$ u_t + u u_x = (0.01/\pi) u_{xx}$$
$$x \in [-1,1],\ \ \ t \in [0,1]$$
$$u(0,x) = -\sin(\pi x)$$
$$u(t,-1) = u(t,1) = 0$$
